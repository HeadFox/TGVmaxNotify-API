# TGVmaxNotify-API
API of the TGVmaxNotify app

## Features
- Create user account
  - city
  - usual journey
- Add new alert trip
  - departure station
  - departure date (prefer)
  - arrival station
  - back date (prefer)
  - mail alert (when available)
  - (smartphone alert) ?
- Ticket history
- Total killometers
